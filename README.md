# 💄 Maquiagem Criativa para Conteúdo Digital

Projeto desenvolvido como desafio prático da DIO utilizando Inteligência Artificial para criação de um ebook visual.

---

## 📖 Sobre o projeto

Este projeto apresenta o processo de criação de um **ebook sobre maquiagem criativa aplicada à criação de conteúdo digital**.

A proposta do material é mostrar como **referências visuais do cinema, games, moda e estética digital podem ser reinterpretadas em looks autorais**, criando identidade visual para redes sociais.

O ebook explora conceitos como:

- inspiração visual
- interpretação estética
- identidade visual
- maquiagem aplicada a conteúdo digital
- construção de estilo autoral

---

## 🤖 Ferramentas de IA utilizadas

As seguintes ferramentas de Inteligência Artificial foram utilizadas na criação do projeto:

- **ChatGPT** – estruturação do conteúdo e desenvolvimento de prompts  
- **Krea AI** – geração de imagens e elementos visuais  
- **Nano Banana AI (versão gratuita)** – criação de imagens conceituais  

Essas ferramentas foram usadas para:

- geração de ideias criativas
- criação de imagens artísticas
- desenvolvimento de prompts
- apoio na estruturação do ebook
